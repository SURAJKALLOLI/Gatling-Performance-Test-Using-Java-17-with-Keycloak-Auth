# Introduction 
Scenario based testing enabler using Gatling.

# Getting Started
1.	Installation process
    Install gatling by referring https://gatling.io/docs/gatling/tutorials/installation/ 
2.	Software dependencies
    Java Version 8,11 or 17.

# Build and Test
On windows, Open "Command Prompt" window. 
Change to ..\PerfTest\bin directory.
Run gatling_scorpio_subscription.bat
1- Press 1 to choose "[1] Run the Simulation locally" option.
2- In "Choose a simulation number" to run the scenario contained in the file.
3- Enter description for the test
